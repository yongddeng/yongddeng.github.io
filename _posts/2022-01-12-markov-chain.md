# Markov Chain
---
Martingale property denotes that the expectated value of the future event equals to the current value given all the past events. In contrast, The Markov property denotes that the future is ind. to the past. For example, weather forecast depends primarily on the current weather instead of the entire history. Probability theory의 기본 목표 중 하나인 stablisation을 mc로 formulate한 뒤 iteration을 돌리는 방식으로도 이룰 수 있다. 


## I
---
A stochastic process $X = (X\_t)\_{t \in T}$ consists of $\mathbb{S}$-valued random variables $X\_{t}:\Omega \to S$ is called a [Markov process]() if it satisfies the [Markov property]() $P(X\_{k} = x\_{k} \,\vert\, X\_{k-1} = x\_{k-1}, \dots, X\_{1} = x\_1) = P(X\_{k} = x\_{k} \,\vert\, X\_{k-1} = x\_{k-1})$. We chiefly use a discrete-time Markov process on a countable state space $\mathbb{S}$, called a [Markov chain](), which the property allows
its joint pmf for $k$ consecutive realisations to be calculated by $P(X\_{k}, X\_{k-1}, \dots, X\_{1}) = P(X\_{k}\,\vert\,X\_{k-1}) P(X\_{k-1}\,\vert\,X\_{k-2}) \dots P(X\_{1}) = (\prod\_{l=2}^{k}P(X\_{l}\,\vert\,X\_{l-1})) P(X\_{1})$. Thus our focus lies on the initial [state probabilities]() $\pi\_{j}(1) = P(X\_{1} = j)$ and the $1$-step [transition probabilities]() $p\_{ij}(1) = P(X\_{k} = j \,\vert\, X\_{k-1} = i)$, where $i, j \in \mathbb{S}$. Nevertheless, many results for Markov chains can be extended to chains with an uncountable state space, so-called Harris chains.

If $\mathbb{S}$ contains $n$ distinct elements (i.e. finite), then we draw a [transition matrix]() $\mathbf{P} = (p\_{ij})$ with $\Sigma\_{j \in \mathbb{S}}p\_{ij} = 1$, where $\mathbf{P}$ is an $n \times n$ positive semidefinite matrix. If, in addition, it is [time-homogeneous]() $P(X\_{k} = i \,\vert\, X\_{k-1} = j) = P(X\_{k-1} = i \,\vert\, X\_{k-2} = j)$ for all $k$, the [Chapman-Kolmogorov equations]() yields  $\mathbf{P}(k)\mathbf{P}(s) = \mathbf{P}(k+s) = [p\_{ij}(k+s)]$ for all $k, s \in T$. Therefore, the $k$-th power of transition matrices is equal to the $k$-step transition matrix $\mathbf{P}^k = \mathbf{P}(k) = [p\_{ij}(k)]$ **(#1)**. For instance, let $k=2$ and $h \in \mathbb{S}$ be an intermediate state among $i$ and $j$, then we get $p\_{ij}(2) = P(X\_{k} = j \,\vert\, X\_{k-2} = i) = \Sigma\_{h}P(X\_{k}=j \,\vert\, X\_{k-1}=h) P(X\_{k-1}=h \,\vert\, X\_{k-2}=i) = \Sigma\_{h}p\_{ih}(1)p\_{hj}(1)$.

We now consider a state pmf $\pi(k) = [\pi\_{j}(k)]$, where $\pi\_{j}(k) = \Sigma\_{i}p\_{ij}\pi\_{i}(k-1)$. If $\pi\_{j}(k)\to\pi\_{j}$ and $\pi\_{i}(k-1)\to\pi\_{i}$ as $k\to\infty$, then $\lim\_{k\to\infty}\pi(k) = \pi = (\pi\_{j})$ is a stationary state pmf or a [steady state](https://www.youtube.com/watch?v=4sXiCxZDrTU), where $\pi$ is a $1 \times n$ row vector with $\Sigma\_{j}\pi\_{j} = 1$ and $\pi\_{j} = \Sigma\_{i}p\_{ij}\pi\_{i}$. The [Perron-Frobenius theorem]() enumerates the eigenvalues of $\mathbf{P}$ by $1= \vert \lambda\_{1} \vert > \vert \lambda\_{2} \vert \geq\cdots\geq \vert \lambda\_{n} \vert$ and its matrix notation $\pi\mathbf{P}=\pi$ is in the form $v^{\top}A = v^{\top}\lambda$. That is, the left eigenvector $q\_{1}$ which corresponds to $\lambda\_{1}$ and satisfies $q\_{1}^{\top}\mathbf{P} = q^{\top}\_{1}\lambda\_{1}$ is a steady state, and the rate of convergence for $\pi(k)$ is in the order of $\lambda_2/\lambda_1$. The limiting distribution $\pi=\pi(1)\mathbf{P}^{\infty}$ of a time-homogeneous Markov chain (if exists) is always unique (which does not need to be true for a general Markov process).


## II
---
The followings are jargons that are taken in the formal statements describing behaviours of a Markov chain and the conditions under which a state pmf becomes steady. A state $j$ is [accessible]() from a state $i$, denoted by $i \rightarrow j$, if $p\_{ij}(k) > 0$ for some $k\geq{1}$. Whereas, $i$ is [absorbing]() if $p\_{ij}(k) = 0$ for some $k\geq{1}$ and all $j \in S$. If both $i \rightarrow j$ and $j \rightarrow i$, and thus $i \leftrightarrow j$, we say $i$ and $j$ [communicate]() with each other **(#2)** and belong to the same [class](). A collection of all states of a Markov chain will be consisting of one or more disjoint communication classes. A Markov chain with a single class is [irreducible]() in which every states in the chain communicate with each other (i.e. a state space is the class).

Suppose $f\_{ii} = P(X\_{k} = i \;\text{for some}\; k>1 \,\vert\, X\_{1}=i)$ is a probability that a system departed from $i$ returns to $i$ for all $i \in \mathbb{S}$. We say $i$ is [recurrent]() if $f\_{ii} = 1$, and [transient]() if $f\_{ii} < 1$. That is, a recurrent state occurs infinitly often and a transient state occurs finitly often. In fact, we can assume that a probability of escaping a transient state $i$ follows $\operatorname{Bernoulli}(1-f\_{ii})$, so that a number of returns to $i$ during a $k$-steps follows $\text{Geom}(1-f\_{ii})$. Whereas, if we let $\Sigma\_{k=2}^{\infty}p\_{ii}(k) = \Sigma\_{k=2}^{\infty}\operatorname{E}[I\_i(X\_{t\_k})\,\vert\,X\_{t\_{1}}=i] = \operatorname{E}[\Sigma\_{k=2}^{\infty}I\_{i}(X\_{t\_k})\,\vert\,X\_{t\_1}=i]$ be a mean number of returns to a state $i$ **(#3)**, then $i$ is recurrent if and only if $\Sigma\_{k=2}^{\infty}p\_{ii}(k) = \infty$ and transient if and only if $\Sigma\_{k=2}^{\infty}p\_{ii}(k) < \infty$.

Provided that, we say $i$ has [period]() $d$ if $p\_{ii}(k) = 0$ whenever $k$ % $d \neq 0$ for all $k > d$ **(#4)**. A state $i$ is [periodic]() if $d>1$, and [aperiodic]() if $d=1$. Note that, period, like recurrence and transience, is a class property. Hence, every state in a class has the same period, and a Markov chain is aperiodic if it is irreducible with period $d=1$. Subsequently, we let $\mu\_{ii} = \operatorname{E}t\_{ii}$ be a [mean recurrence time](), where $t\_{ii}$ is an elapsed time to returning $i$ from $i$, and say $i$ is [positive recurrent]() if $\mu\_{ii} < \infty$, and [null recurrent]() if $\mu\_{ii} = \infty$. If a recurrent state $i$ is the initial departure and $\lbrace t\_{ii}(l) \rbrace\_{1 < l \leq k}$ is a sequence of i.i.d. elapsed times, then a proportion of time spent in $i$ until $l$-th returns is given by $l/\Sigma\_{m=1}^{l}t\_{ii}(m)$ such that $l/\Sigma\_{m=1}^{l}t\_{ii}(m) \to 1/\mu\_{ii} = \pi\_i$ as $k\to\infty$ due to the WLLN.

A state $i$ is [ergodic](https://stats.libretexts.org/Bookshelves/Probability_Theory/Book%3A_Introductory_Probability_(Grinstead_and_Snell)/11%3A_Markov_Chains/11.03%3A_Ergodic_Markov_Chains) if it is aperiodic and positive recurrent. If all states in an irreducible Markov chain are ergodic, then it is called an ergodic Markov chain. In particular, a Markov chain is ergodic if there is a number $N$ such that any state can be reached from any other state in any number of steps less or equal to a number $N$. A fully connected transition matrix (i.e. all transitions have a non-zero probability) holds the sufficiency with $N = 1$.
