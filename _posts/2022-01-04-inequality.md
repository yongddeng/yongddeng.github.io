# Inequality
---
Queried: Who invented inequalities; Google responded: The signs for greater than and less than were introduced by Thomas Harriot (1560-1621), but he initially wrote triangular symbols, and the editor of his book altered to what resemble the modern symbols. Inequalities in probability thoery are essential in order to adapt some mathematics originated in geometry and linear algebra.


## I
---
Suppose $X:\Omega \to \mathbb{R}$ is a random variable and $g:\mathbb{R} \to [0,\infty]$ is a (Borel) measurable function. If we let $g\_*(A) = \inf \lbrace g(y): y \in A \rbrace$ for any $A \in \mathcal{B}\_{\mathbb{R}}$, then [Markov's inequality](https://en.wikipedia.org/wiki/Markov%27s_inequality) states that $g_*(A)P(X \in A) \leq \operatorname{E}g(X)I\_{X \in A} \leq \operatorname{E}g(X)$, where $P(X \in A) = \operatorname{E}I\_{X \in A}$. As the same instance, if we let $g(x) = x^+$ and $A = [a,\infty)$ for some $a > 0$, then $P(X \geq a) \leq a^{-1}\operatorname{E}X^+$, and so it is particularly appealing for nonnegative $X$. Furthermore, if $g(x) = {\vert x \vert}^q$ and $A = (-\infty, a] \cup [a, \infty)$ for some $a > 0$, then $P(\vert X \vert \geq a) \leq a^{-q}\operatorname{E}{\vert X \vert}^q$. In particular, by letting $q=2$ and $X = Y - \operatorname{E}Y$ for $Y \in L^2$, we would be able to construct a much tighter upper bound $P(\vert Y - \operatorname{E}Y \vert \geq a) \leq a^{-2}\operatorname{Var}Y$, so-called [Chebyshev's inequality](https://en.wikipedia.org/wiki/Chebyshev%27s_inequality) **(#1)**. 

The second instance also works for a sequence $(X\_n)\_n\in\mathbb{N}$ of independent random variables with $\operatorname{E}X_n = 0$ and $\operatorname{Var}X_n < \infty$. Formally, suppose $S\_n = \Sigma\_{k=1}^n X\_k$, then $P(\vert S\_n - \operatorname{E}S\_n \vert \geq a) \leq a^{-2}\operatorname{Var}S\_n$, where $\operatorname{Var}S\_n = \Sigma\_{k=1}^n\operatorname{Var}X\_k$. That is, if $(X\_n)\_n\in\mathbb{N}$ are i.i.d., then $P(\vert S_n - \operatorname{E}S_n \vert \geq a) \leq a^{-2} n\operatorname{Var}X_1$. If necessary, [truncated Chebyshev's inequality]() can drop the finite variance assumption via [truncation]() **(#2)**. Suppose $\tilde{X}\_n = X_n - \operatorname{E}X_n$ only with $X_n \in L^2$, then $P(\vert \tilde{X}_n \vert \geq a) \leq a^{-2}\operatorname{Var}\tilde{X}_n$. Hence by letting $\tilde{S}_n = \Sigma_{k=1}^n \tilde{X}_k$, we conclude that $P(\vert S - \operatorname{E}\tilde{S}_n \vert \geq a) \leq a^{-2} \Sigma_{k=1}^n\operatorname{Var}\tilde{X}_k + \Sigma_{k=1}^{n} P(\vert X_k \vert > b_k)$, where $\tilde{X}_k = X_k$ when $\vert X_k \vert \leq b_k$ and $\tilde{X}_k = c$ otherwise (we assumed $b\_k$ is a positive real and $c$ is $0$).

Chebyshev's inequality might be altered to [Kolmogorov's inequality](https://en.wikipedia.org/wiki/Kolmogorov%27s_inequality) (i.e. maximal inequality) assuring that $P(\max_{1 \leq k\leq n} \vert S_k \vert \geq a) \leq a^{-2}\operatorname{Var}S_n$, or, perhaps $P(\max_{n \leq k\leq m} \vert S_k \vert \geq a) \leq a^{-2}\operatorname{Var}S_m$. Its convenience arises when we need to bound the worst case deviation (that can occure at any $k$) of a partial sum process $(S_n)_{n\in\mathbb{N}}$ consists of $S_n = \Sigma_{k=1}^{n}X_k$, where each $X_k$ are pairwise independent. This can be further generalised to [Hájek–Rényi inequality](https://www.jstor.org/stable/25049653) which states that $P(\max\_{1 \leq k \leq n} d_k \vert S_k \vert \geq a) \leq a^{-2}\Sigma_{k=1}^{n} d_{k}^{2}\operatorname{Var}X_k$, where $(d_k)\_{1 \leq k\leq n}$ are positive non-increasing reals. With $d_k =1$ for all $k$, it is simply the maximal inequality. Note that all inequalities have been mentioned so far take a value in the tails of the distribution.


## II
---
Some inequalities rather relate moments of sums to sum of moments. Suppose $X$ and $Y$ are random variables on the same probability space such that $\operatorname{E}{\vert X \vert}^r, \operatorname{E}{\vert Y \vert}^r < \infty$ for some $r > 0$. The [$c_r$-inequality](https://jeanmariedufour.github.io/ResE/Dufour_2008_C_TS_Moments.pdf) assures that $\operatorname{E}{\vert X+Y \vert}^r \leq c_r(\operatorname{E}{\vert X \vert}^r + \operatorname{E}{\vert Y \vert}^r)$, where $c_r = 1$ whenever $r \leq 1$ and $c_r = 2^{r-1}$ whenever $r \geq 1$. In particular, for all $r \in (0,1]$, this can be seen as the [triangle inequality](https://en.wikipedia.org/wiki/Triangle_inequality) of the $r$-th order that assures $\operatorname{E}{\vert X+Y \vert}^r \leq \operatorname{E}\vert X \vert + \vert Y \vert)^r \leq \operatorname{E} {\vert X \vert}^r + \operatorname{E}{\vert Y \vert}^r$. Therefore, the sum of random variables are integrable if every summands are integrable. In fact, the converse holds for all $r > 0$ if and only if the random variables $X$ and $Y$ are independent, and the $c_r$-inequality can be mainly used to prove the statement.

Akin to the last two inequalities, [Minkowski's inequality](https://en.wikipedia.org/wiki/Minkowski_inequality) concerns sums of random variables in an $L^p$-space by using the [$L^p$-norm](https://en.wikipedia.org/wiki/Norm_(mathematics)) that we define by ${\left\lVert X \right\rVert}_p = (\operatorname{E} {\vert X \vert}^p)^{1/p}$. Specifically, for any $p \geq 1$, if one has $\operatorname{E}{\vert X \vert}^p, \operatorname{E}{\vert Y \vert}^p < \infty$, then it assures that ${\left\lVert X+Y \right\rVert}_p \leq {\left\lVert X \right\rVert}_p + {\left\lVert Y \right\rVert}_p$. In fact, well-defined norms ${\left\lVert X \right\rVert}_p, {\left\lVert Y \right\rVert}_p \in L^p$ yields ${\left\lVert \alpha{X} + \beta{Y} \right\rVert}_p \in L^p$ for all $\alpha,\beta \in \mathbb{R}$. Moreover, since the operator $\left\lVert \cdot \right\rVert: \mathbb{R} \to [0,\infty)$ holds linearity by ${\left\lVert \alpha{X} \right\rVert}_p = \vert \alpha \vert {\left\lVert X \right\rVert}_p$ for all $\alpha \in \mathbb{R}$, an $L^p$-space can be viewed as a vector space **(#3)**. In particular, if every sequence is a Cauchy sequence, it is a Banach space. If, in addition, we adapt the covariance that is essentially an inner product, then a Banach space evolves into a Hilbert space $H$. 

While Minkowski's inequality concerns sums, [Hölder's inequality](https://en.wikipedia.org/wiki/H%C3%B6lder%27s_inequality) concerns products. The latter states that, if $p, q > 1$ and $p^{-1} + q^{-1} = 1$ **(#4)**, then $\vert\langle X,Y \rangle\vert := {\vert \operatorname{E}XY \vert} \leq {\operatorname{E}\vert XY \vert} \leq {\left\lVert X \right\rVert}_p \cdot {\left\lVert Y \right\rVert}_q$. The second inequality makes sense once we observe that $\operatorname{E}({\vert X \vert}/{\left\lVert X \right\rVert}_p \cdot {\vert Y \vert}/{\left\lVert Y \right\rVert}_p) \leq p^{-1}\operatorname{E}({\vert X \vert}^p/{\left\lVert X \right\rVert}_p^p) + q^{-1}\operatorname{E}({\vert Y \vert}^q/{\left\lVert Y \right\rVert}_q^q) = p^{-1} + q^{-1}$. If we let $p=q=2$, then Hölder's inequality turns into [Cauchy-Schwarz ineqaulity]() that assures $\operatorname{E}{\vert XY \vert} \leq {\left\lVert X \right\rVert}_2 \cdot {\left\lVert Y \right\rVert}_2 = \sqrt{\operatorname{E}X^2\cdot\operatorname{E}Y^2}$. Also, slight modifications bounds ${\vert \operatorname{Cov}(X,Y) \vert} \leq \sqrt{\operatorname{Var}X \cdot \operatorname{Var}Y}$ and gives perhaps one of the most popular measure of statistics so-called the [Pearson correlation coefficient](https://en.wikipedia.org/wiki/Pearson_correlation_coefficient) ${\vert \rho\_{X,Y} \vert} \leq 1$, where $\rho\_{X,Y} = \operatorname{Cov}(X,Y) / \sqrt{\operatorname{Var}X \cdot \operatorname{Var}Y}$.


## III
---
[Jensen's inequality](https://en.wikipedia.org/wiki/Jensen%27s_inequality) states that a secant line which consists of weighted means of a convex function $\varphi$ with the total weight $1$ lies above the graph of $\varphi$. If we let $(\Omega, \mathcal{F}, P)$ be a probability space, $X$ be an $\mathbb{S}$-valued integrable random variable, and $\varphi:\mathbb{R}\to\mathbb{R}$ be a measurable convex function, then Jensen's inequality assures that $\varphi(\operatorname{E}(X\vert\mathcal{G})) \leq \operatorname{E}(\varphi(X)\vert\mathcal{G})$, where $\mathcal{G} \subset \mathcal{F}$ is a sub-$\sigma$-algebra. When a set $\mathbb{S}$ is the real axis $\mathbb{R}$, and $\mathcal{G}$ is the trivial σ-algebra containing $\{\Omega, \emptyset\}$, then one has $\varphi(\operatorname{E}X) \leq \operatorname{E}\varphi(X)$, where "$=$" comes if and only if $\varphi$ is a linear function on a [convex set](https://en.wikipedia.org/wiki/Convex_set) $A \subseteq \mathbb{R}$ such that $P(X \in A) = 1$ **(#5)**. Here we interrupt ourselves to browse some applications of Jensen's inequality and Cauchy-Schwarz inequality.

The direct use of Jensen's inequality on a density function $f$ gives an upper bound $\varphi(\int_{\mathbb{R}} g(x)f(x)\,\mathrm{d}x) \leq \int_{\mathbb{R}} \varphi(g(x))f(x)\,\mathrm{d}x$, where $g$ is a Borel measurable function (i.e. $g(x) = x$). In particular, if $\varphi(x) = x^{2n}$ for some $n \in \mathbb{N}$, then $\varphi^{\prime\prime}(x) \geq 0$ for any $x\in\mathbb{R}$, and thus $\varphi(\operatorname{E}X) = (\operatorname{E}X)^{2n} \leq \operatorname{E}X^{2n}$. It immediately  relates to the non-negativity of $\operatorname{Var}X = \operatorname{E}X^2 - (\operatorname{E}X)^2 \geq 0$. Furthermore, let $p$ and $q$ be the true and another density of $X$, respectively. If we set $Y(x) = q(x)/p(x)$ and $\varphi(y) = -\log(y)$ **(#6)**, then Jensen's inequality yields [Gibbs' inequality](https://en.wikipedia.org/wiki/Gibbs%27_inequality) $-\Sigma_{x}p(x)\log{p(x)} \leq -\Sigma_{x}p(x)\log{q(x)}$, which in trun relates to the non-negativity of the [KL-Divergence](https://en.wikipedia.org/wiki/Kullback%E2%80%93Leibler_divergence) $D_{KL}(P \vert\vert Q) = -\Sigma_{x}p(x) \log{{q(x) \over p(x)}} \geq 0$ ($:= \Sigma_{x}p(x) \log{{p(x) \over q(x)}}$).

Since our notions of length and orthogonality are preserved within a set $H = \{X: \operatorname{E}X^2 < \infty\}$ we can work with $X \in H$ geometrically as if they were vectors in Euclidean space. For example, given $X,Y \in H$, the OLS of $Y$ can be done by finding the best linear function of $X$. Subsequently, $L(Y \vert X) = \operatorname{E}Y + (X-\operatorname{E}X) \cdot {\operatorname{Cov}(X,Y) \over \operatorname{Var}X}$ is the best function uniquely satisfying (i) $\operatorname{E}(L(Y \vert X)) = EY$; (ii) $\operatorname{Cov}(X, L(Y \vert X)) = \operatorname{Cov}(X,Y)$; If $U \in H$ is any linear function of $X$, then equality holds for $\operatorname{E}[Y - L(Y \vert X)]^2 \leq \operatorname{E}(Y-U)^2$ if and only if $U = L(Y \vert X)$. Whilst SGD methods are often used in training phase under the convexity, the mse can be mathematically presented by $\operatorname{E}[Y - L(Y \vert X)]^2 = \operatorname{Var}(Y)[1-\operatorname{Cor}^2(X,Y)]$.


## IV
---
We end up in reading inequalities related to the median which we can prove the existence but not the uniquness. The [median](https://en.wikipedia.org/wiki/Median) $\operatorname{Med}X$ of a random variable $X$ is real-valued numbers that satisfy the inequality $P(X \leq \operatorname{Med}X) \geq 1/2 \land P(X \geq \operatorname{Med}X) \geq 1/2$ **(#7)**. Accordingly, if $P(\vert X \vert > a) < 1/2$ for some $a \in \mathbb{R}^{+}$, then ${\vert \operatorname{Med}X \vert} \leq a$. Moreover, if $\operatorname{Var}X < \infty$, then the gap between $\operatorname{Med}X$ and $\operatorname{E}X$ is bounded by ${\vert \operatorname{Med}X - \operatorname{E}X \vert} = {\vert \operatorname{E}(\operatorname{Med}X - X) \vert} \leq \operatorname{E}{\vert X - \operatorname{Med}X \vert} \leq \operatorname{E}{\vert X - \operatorname{E}X \vert} \leq \sqrt{\operatorname{E}{\vert X - (\operatorname{E}X)^2 \vert}}$, where $\sqrt{\operatorname{E}{\vert X - (\operatorname{E}X)^2 \vert}} = \sqrt{\operatorname{Var}X}$. The first and the third inequalities are due to Jensen's inequality being applied with each convex functions $\varphi(x) = \vert x \vert$ and $\varphi(x) = x^2$.

Besides, we say $X$ is [symmetric around $a$](https://www.planetmath.org/symmetricrandomvariable) if two shifted random variables $X-a$ and $a-X$ have the same distribution $F_{X-a} = F_{a-X}$ for all $x\in\mathbb{R}$. We can also say that $X$ is symmetric around $a$ if $P(X \geq a+x) = P(X \leq a-x)$ for all $x \in \mathbb{R}$ due to the fact that $F_{X-a} = F_{a-X}$ if and only if $f_{X-a}(a+x) = f_{X-a}(a-x)$. Then, we often say $X$ is [symmetric](http://dept.stat.lsa.umich.edu/~moulib/Homework2-fall05sol.pdf) if $F_X = F_{-X}$ for all $x\in\mathbb{R}$, or, equivalently, $f_{X}(x) = f_{X}(-x)$ for all $x\in\mathbb{R}$. In fact, if $X$ is symmetric, then one can always obtain a finite $\operatorname{E}X$ such that $\operatorname{E}X = \operatorname{E}(-X) = -\operatorname{E}X$. If, in addition, $X$ is unimodal, then $\operatorname{E}X = \operatorname{Med}X = \operatorname{Mode}X$, or, if $X$ is bimodal, then $\operatorname{E}X = \operatorname{Med}X$. Hence it may be easier to prove theorems with symmetric $X$ then take relaxations.

Whenever $X$ is non symmetric, using another ind. random variable $X^\prime \stackrel{d}{=} X$ **(#8)**, we can define a symmetrised random variable $X^s = X - X^\prime$ whose $\operatorname{E}X^s = 0$. Subsequently, for every $a, b \in \mathbb{R}$, the [weak symmetrisation inequalities](https://books.google.co.kr/books?id=9TmRgPg-6vgC&pg=PA134&lpg=PA134&dq=strong+symmetrisation+inequalities&source=bl&ots=R5zmAxXEqz&sig=ACfU3U10RulyGkz9LqpvwXoR1jsEu52qvQ&hl=en&sa=X&ved=2ahUKEwjbo8P-rqX2AhUPZ94KHfWaAS4Q6AF6BAgsEAM#v=onepage&q=strong%20symmetrisation%20inequalities&f=false) assure that $P({\vert X - \operatorname{Med}X \vert} \geq b) \leq 2P(X^s \geq b) \leq 4P({\vert X - \operatorname{Med}X \vert} \geq b/2)$. Also, for all $(a_n)_n\in\mathbb{N}$ and $b$ on $\mathbb{R}$, the [strong symmetrisation inequalities]() assures $P(\max\_{1 \leq k \leq n}{\vert X_k - \operatorname{Med}X_k \vert} \geq b) \leq 2P(\max\_{1 \leq k \leq n}{\vert X_k^s \vert} \geq b) \leq 4P(\max\_{1 \leq k \leq n}{\vert X_k - \operatorname{Med}X_k \vert} \geq b/2)$. Note that, the relations between the two inequalities are akin to that of Chebyshev's and Kolmogorov's, and we can extend the use of the median with symmetrisation to bound sums of random variables.
