# Probability Theory
---

Next, I perused Amir Dembo's [Stanford STAT310](https://web.stanford.edu/class/stats310a/lnotes.pdf) and Allan Gut's [Probability: A Graduate Course](https://www.usb.ac.ir/FileStaff/5678_2018-9-18-12-55-51.pdf) along with [W1](https://www.randomservices.org), [W2](http://theanalysisofdata.com/), [W3](https://www.stat.berkeley.edu/~aldous/) to acquire the modern measure-theoretic foundations of probability theory.


## I
---
A [probability theory](https://en.wikipedia.org/wiki/Probability_theory) is grounded in the framework of random [experiments](https://en.wikipedia.org/wiki/Experiment_(probability_theory)) (i.e. trials) where the outcome is inherently uncertain prior to execution. Practical execution allows for repetition under consistent or analogous conditions, temporally or spatially **(#1)**. It can be contended that the significance lies in the confidence derived from such repetitions superseding our a priori knowledge. The subject matter pertains to fully specified mathematical objects, in contrast to [statistics](https://en.wikipedia.org/wiki/Statistics), which deals with incomplete ones. We instead engage in the drawing, analysis, and visualisation of samples to determine an unbiased estimator for one or more unknown parameters in the latter.


Let a set $\Omega$ be a [sample space](https://en.wikipedia.org/wiki/Sample_space), an element $\omega \in \Omega$ be an [elementary event](https://en.wikipedia.org/wiki/Elementary_event), and a subset $A \subseteq \Omega$ be an [event](https://en.wikipedia.org/wiki/Event_(probability_theory)) such that $\lvert{A}\rvert \leq \lvert{\Omega}\rvert$. While we say $A$ is occurred (not occurred) if $\omega \in A$ ($\omega \notin A$), let a [$\sigma$-algebra](https://en.wikipedia.org/wiki/%CE%A3-algebra) $\mathcal{F} = \sigma(A)$ such that (i) $\Omega \in \mathcal{F}$; (ii) if $A \in \mathcal{F}$, then $A^{c} \in \mathcal{F}$; (iii) if $(A\_{n})\_{n \in \mathbb{N}} \in \mathcal{F}$, then $\bigcup_{n \in \mathbb{N}} A_n \in \mathcal{F}$; There may exist multiple $\sigma$-algebras $\mathcal{F}^* = \lbrace \mathcal{F}_0, \mathcal{F}_1, \dots \lvert \, A \subset \mathcal{F_n} \rbrace$, and we say $\mathcal{F}$ is countably generated if there exists only countably many generators **(#2)**. Loosely speaking, a $\sigma$-algebra generated by $A$ quantifies the amount of information that can be extracted from $A$, thus having the largest possible $\sigma$-algebra would result a more friutful tuple $(\Omega, \mathcal{F})$. 

The [power set](https://en.wikipedia.org/wiki/Power_set) $\mathcal{F}\_\Omega = \sigma(\Omega)$ can be taken for a finite or a countable $\Omega$. This however violates the below axiom if we had uncountable $\Omega$ (i.e. $\mathbb{R}$) as a sum of uncountably many probabilities $P(\Omega) = \Sigma\_{\omega\in\Omega} p\_\omega = \infty$. To remedy this, for example, we can take the smallest $\sigma$-algebra containing every open subsets (i.e. that are indeed measurable) called the [Borel $\sigma$-algebra](https://en.wikipedia.org/wiki/Borel_set) $\mathcal{B}\_{\mathbb{R}} = \sigma(\mathcal{O} \subset \mathbb{R})$ and build an Euclidean space $(\mathbb{R}^n, \mathcal{B}\_{\mathbb{R}^n})$. Besides, the monotone class theorem and the $\pi - \lambda$ theorem exist for an algebra and a $\pi$-system, respectively. If an algebra $A$ is a subset of a monotone class $\mathcal{M}$ **(#3)**, then the smallest $\sigma$-algebra $\sigma(A)$ is percisely equal to the smallest monotone class $\mathcal{M}(A)$.


## II
---
A [probability measure](https://en.wikipedia.org/wiki/Probability_measure) $P: \mathcal{F} \to [0,1]$ defined on $(\Omega, \mathcal{F})$ should satisfy the following [Kolmogorov axioms](https://www.stat.berkeley.edu/~aldous/Real_World/kolmogorov.html) (i) $P(A) \geq 0$ for any $A \in \mathcal{F}$; (ii) $P(\Omega) = 1$; (iii) if $(A\_{n})\_{n\in\mathbb{N}}$ is a set of pairwise disjoint events, then $P(\bigcup\_{n=1}^{\infty} A\_{n}) = \Sigma_{n=1}^{\infty} P(A_n)$ **(#4)**; As expected, it inherits many useful properties of a measure such as (i) zero probability: $P\left(\emptyset\right) = 0$; (ii) complement: $P(A^c) = 1 - P(A)$; (iii) monotonicity: if $A \subseteq B$, then $P(A) \leq P(B)$; (iv) sub-additivity: if $A \subseteq \bigcup_n A_n$, then $P(A) \leq \Sigma_n P(A_n)$; and so on. The monotonicity especially allows $P$ to be well-defined on the limits (i.e. if exists) of a sequence $(A_n)_{n\in\mathbb{N}}$ of events. Note that there are non-empty sets with zero probability.

If we properly assign a probability $p_\omega \geq 0$ for every $\omega \in \Omega$, making sure that $P\left(\Omega\right) = \Sigma_{\omega\in\Omega}\, p_{\omega} = 1$, then a probability of $A$ occurring is defined by $P(A) = \Sigma_{\omega\in{A}}\, p_\omega$. For instance, if we assign the [uniform probability](https://en.wikipedia.org/wiki/Continuous_uniform_distribution) $p_\omega = {\lvert \Omega \rvert}^{-1}$ on some finite $\Omega \subset \mathbb{R}$, then $P$ will behave simply as the counting measure. Whereas, the [Gaussian probability](https://en.wikipedia.org/wiki/Normal_distribution) $f(x) = {1 \over \sigma\sqrt{2\pi}} e^{-{1 \over 2}({x - \mu \over \sigma})^2}$ draws a more interesting shape which can also describe the nature. In discrete sense, given a set of non-negative integers $\Omega = \lbrace k: k \in \mathbb{N}\_{0} \rbrace$ and an expected rate of occurrences $\lambda > 0$, the [Poisson probability](https://en.wikipedia.org/wiki/Poisson_distribution) $p_k = {\lambda^k\over{k!}}e^{-\lambda}$ which can be derived from the [binomial distribution](https://en.wikipedia.org/wiki/Binomial_distribution) assigns measures to an event occurring $k$ times.

A triple $(\Omega, \mathcal{F}, P)$, which we call a [probability space](https://en.wikipedia.org/wiki/Probability_space), shall be restricted to a [complete probability space](https://math.stackexchange.com/questions/4095399/complete-probability-spaces) for the measurability of null sets. If there exists $B \in \mathcal{F}$ such that $A \subset B$ with $P(B) = 0$ (i.e. $B$ is measurable), then $A \subset \Omega$ which may or may not be an element of $\mathcal{F}$ is a [null set](g/wiki/Null_set), and here the completion treats $A$ being measurable whenever $B$ is measurable. On the other hand, if there does not exists $\psi \in \Omega$ such that $P(\psi) > P(\omega) > 0$, then $\omega \in \Omega$ is an atom **(#5)**. It hints us that an uncountable $\Omega$ can make a [non-atomic proability space](https://en.wikipedia.org/wiki/Atom_(measure_theory)). I.e. the counting measure $\mu$ on $\Omega = \lbrace 1,2,\dots, 10 \rbrace$ takes a singleton $\lbrace j \rbrace$ as an atom, but the Lebesgue measure $\lambda$ on $\Omega = \mathbb{R}$ has no atom.


## III
---
Suppose $(A\_{n})\_{n \in \mathbb{N}}$ and $A$ are subsets of $\Omega$. If $(A\_{n})\_{n\in\mathbb{N}}$ is [strictly increasing](https://en.wikipedia.org/wiki/Monotonic_function) (strictly decreasing), i.e. if $A\_{n} \subset A\_{n+1}$ ($A\_{n} \supset A\_{n+1}$), then $\lim\_{n \to \infty} A\_{n} = \bigcup\_{n=1}^{\infty} A\_{n}$ ($\lim\_{n \to \infty} A\_n = \bigcap\_{n=1}^{\infty} A\_{n}$). The limit of increasing unions of the initial segments $\bigcup\_{k=1}^{n} A\_{k} \subset \bigcup\_{k=1}^{n+1} A\_k$ is given by $\lim\_{n \to \infty} \bigcup\_{k=1}^{n} A\_k = \bigcup\_{k=1}^{\infty} A\_k$, and the limit of decreasing unions of the tail segments $\bigcup\_{k=n}^{\infty} A\_{k} \supset \bigcup\_{k=n+1}^{\infty} A\_{k}$ is given by $\lim\_{n\to\infty} \bigcup\_{k=n}^{\infty} A\_{k} = \bigcap\_{n=1}^{\infty} \bigcup\_{k=n}^{\infty} A\_{k} = \limsup_{n\to\infty} A\_{n}$. If $\lim\_{n\to\infty} \bigcap\_{k=n}^{\infty}A\_{k} = \bigcup\_{n=1}^{\infty}\bigcap\_{k=n}^{\infty} A\_{k} = \liminf\_{n\to\infty} A\_{n}$ is equal to $\limsup_{n\to\infty} A\_{n}$, then $\liminf_{n\to\infty} A\_{n} = \limsup_{n\to\infty} A\_{n} = \lim\_{n\to\infty} A\_{n} = A$. Notice that the first equality is not true in general.

$P$ on a [bounded monotonic sequence](http://www-groups.mcs.st-andrews.ac.uk/~john/analysis/Lectures/L8.html) can be a continuous function **(#6)** as analogue to a real-valued function with well-defined limits. That is, if $A_n \nearrow A$ ($A_n \searrow A$) as $n \to \infty$, then $P(A_n) \nearrow P(A)$ ($P(A_n) \searrow P(A)$) as $n \to \infty$. In particular, based on the chain of probabilistic inequalities $P\left(\liminf\_{n \to \infty}A\_{n}\right) \leq \liminf\_{n \to \infty} P\left(A\_{n} \right) \leq \limsup\_{n \to \infty} P\left(A\_{n} \right) \leq P\left(\limsup\_{n \to \infty} A\_{n}\right)$, if $A\_{n} \to A$ as $n \to \infty$, then $P\left(A\_{n} \right) \to P(A)$ as $n \to \infty$. The interchange of a probability measure and a limit of a union are also allowed, so we can express $\lim_{n \to \infty} P(\bigcup_{k=1}^{n} A_{k}) = P(\bigcup_{k=1}^{\infty} A_{k})$ and $\lim_{n \to \infty} P(\bigcup_{k=n}^{\infty} A_{n}) = P(\bigcap_{n=1}^{\infty} \bigcup_{k=n}^{\infty} A_{k}) = P(\limsup_{n\to\infty} A\_{n})$.

The [Borel–Cantelli lemmas](https://en.wikipedia.org/wiki/Borel%E2%80%93Cantelli_lemma) state sufficient conditions for a convergence of a sequence $(A\_n)\_{n\in\mathbb{N}}$ of events. Specifically, the 1st lemma states that if $\Sigma\_{n=1}^{\infty} P(A_n) < \infty$, then $P(A_n \text{ i.o.}) = P(\limsup\_{n \to \infty} A_n) \leq \lim_{n\to\infty}\Sigma_{k=n}^{\infty}P(A\_k) = 0$ **(#7)**, signifying that almost every $\omega\in\Omega$ belong to at most a finite number of events $A_n$. Whereas, the 2nd lemma states that if $\Sigma_{n=1}^{\infty} P(A\_n) = \infty$ and the events $A_n$ are independent, then $P(A_n \text{ i.o.}) = 1$. However, desptie of their usabilities in mathematical proofs, the [infinite monkey theorem](https://en.wikipedia.org/wiki/Infinite_monkey_theorem) which Émile Borel brought to the public encountered the pungent criticism, and so the 2nd lemma has been considered to be quite nonsense in practice.
